## Introdução

Projetos realizados no curso de Cientista de Dados da EBAC.

### Objetivo

O curso Cientista de Dados da EBAC tem como objetivo formar profissionais capazes de transformar dados em insights estratégicos para diferentes setores do mercado. Ao longo da formação, são desenvolvidas competências essenciais em Python e SQL, e bibliotecas avançadas como Scikit-learn, Seaborn e Matplotlib, além de fundamentos sólidos em estatística, machine learning e análise exploratória. O programa é estruturado para proporcionar experiência prática com projetos reais, preparando o aluno para atuar no mercado de trabalho.
